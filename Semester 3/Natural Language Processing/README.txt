Prerequisites:
==============
Make sure that the requirements mentioned in requirements.txt is installed, this can be done by running the following:

----------------------------------
 pip install -r requirements.txt
----------------------------------

Make sure that tfidf.py file is in the same path as the app.py file

Running the app:
================
After the prerequisites are met, we need to run the app.py

---------------
 python app.py
---------------

Go to http://127.0.0.1:8000/docs, the end point for the assignment is "/assignment".
